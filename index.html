Introduction

Hold a question in your mind, shake the phone, and an answer of subtle wisdom shall be yours!

This is the first Android application I've written and is largely intended as in introduction to the basics of Android development. Please see the Developer's Guide for more information.

Download

Although currently unavailable on Android Market, you can download the APK file from the GitHub repository, copy it onto a memory card, and install it using App Installer.

Developer's guide

The user experience of the I Ching application is that as they shake their phone then the screen continuously replaces the grey background with black until, after five seconds of cumulative shaking, they are displayed an I Ching hexagram and its respective description. They can then go back and re-shake the phone.

There are two main classes, each with their own Handler objects running in distinct threads:

  • ShakeActivity: At start of day it:
    • draws the grey background and help text that says "Please shake the phone".
    • starts the AccelerometerManager.
  • When its Handler receives MSG_TYPE_START_SHAKING messages it continuously fills the screen with black, and keeps itself going by continuously shaking by sending itself MSG_TYPE_SILL_SHAKING messages. When it has shaken enough it:
    • creates an instance of MersennseTwister and seeds it with AccelerometerManager::getEntropy(), in order to get an integer corresponding to a random integer.
    • starts DivinationActivity with the hexagram's integer, which loads up the correct HTML description.
  • AccelerometerManager: Implements the SensorListener interface and when shaking of sufficient force occurs it sends ShakeActivity's Handler a MSG_TYPE_START_SHAKING message. When it stops shaking it sends a MSG_TYPE_STOP_SHAKING message to ShakeActivity's Handler.
